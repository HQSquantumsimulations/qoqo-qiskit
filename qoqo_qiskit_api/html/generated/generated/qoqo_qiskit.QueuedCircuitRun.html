Parameters:

  • job (Job) -- The job that is run.

  • memory (bool) -- True if the result is meant to be read via job.get_memory() instead of job.get_counts().

  • sim_type (str) -- The simulation type. This can be "automatic", "statevector" or "density_matrix".

  • registers_info (Tuple[Any]) -- The initially setup registers.

  • res_index (Optional[int]) -- The index of the ExperimentalResult in Result.results. Defaults to 0. It can be relevant in case the circuit has been run as part of a list.
